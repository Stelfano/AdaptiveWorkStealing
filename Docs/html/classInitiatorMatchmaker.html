Construct a new Initiator Matchmaker object.

The only initiator in the hierarchy is the root rank which creates a communicator group with all worker nodes, the constructor also inizializes globalResult variable to zero (used for final reduction)

Parameters
parentRankRank of parent node (always -1 since this node is the root of the tree)
chunkSizeSize of the chunk
recvBufferPointer to initial reciever buffer
totalParticlesNumber of total particles (in this case the total number of starting particles)
localAverageLocal Average for root rank (total particles / number of childs)
localThresholdLocal Threshold
childNumberNumber of childs
childRanksRanks of childs
dataCommCommunicator containing all workers and root rank

Function to check termination of computation If lower average is 0 causes all below nodes to recieve 0 average update and thus causing them to initiate termination procedures, only the initiator rank can communicate instatly this value, all other matchmaker are prevented to send 0 value average to avoid early termination of reduction.

Returns
1 if lower average is 0

virtual void InitiatorMatchmaker::updateAverage ( )
inlineprotectedvirtual

This function updates average when metrics from the nodes below are recieved.

This version of the function is the only one that thruthfully communicates real value average to below nodes, to initialize termination this function is able to set lower average value to 0 and the send this average value to all childs, this causes cascade termination
